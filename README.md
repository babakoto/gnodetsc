# Description
Generate nodejs project with typescript + nodemon

### Step 1
`npm i -g gnodetsc `

### Step 2
`gnodetsc init myprojet 8080`

### Step 3 
`cd myprojet` 

### Step 4 : install dependencies
`npm install`  

### Step 5 : run server
`npm run dev`
